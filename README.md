# weather-app

Don't forget to get your "api_key" and add it to your code :) 
https://openweathermap.org/api 


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
